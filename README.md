# Medical Imaging DevOps Platform

This project is a demo-grade imaging DevOps platform that shows how a medical imaging stack can be built, deployed, and operated end to end, combining Orthanc for DICOM storage and routing, a FastAPI backend, PostgreSQL for structured data, MinIO for object storage, monitoring and audit logging, automated backups, and an optional AI component, all wired together for local development and demonstration rather than clinical use.
